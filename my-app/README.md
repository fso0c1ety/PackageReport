# Package Raport (Mobile)

A simple mobile app (Expo React Native) to add report entries with columns: **Eksportusi**, **Importusi**, **KG**, **Data/Ora**, **Produkti**. Starts on an empty page with a plus button to add new columns (entries), and lists them in a table layout.

## Prerequisites
- Node.js 18+
- Expo Go app on your phone (Android/iOS)

## Quick Start
```bash
# From the project folder
npm install
npm run start
```
- Scan the QR code with Expo Go.
- Tap the plus button, fill the fields, and save.
- Entries persist locally on the device.

## Notes
- Data is saved using AsyncStorage.
- This is a single-screen app per your request.
